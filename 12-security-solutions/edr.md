# Endpoint Detection and Response (EDR)

## What is EDR?

EDR is a security solution that constantly monitors endpoint devices. These devices are:

- Desktops
- Laptops
- Mobile devices
- Tablets
- Smart Watches
- IoT Devices
- POS Systems
- Medical Devices
- Digital Printers
- Servers

These solutions perform different operations on endpoint devices to keep device secure.

## EDR Core Components

- Endpoint data collection agents
- Automated response
- Analysis and forensics

## Popular EDR Products

- SentinelOne
