# Intrusion Prevention System (IPS)

## What Is IPS?

Intrusion prevention system detects and prevents breaches by taking the necessary action.

---

## Types Of IPS

### 1. Network-Based Intrusion Prevention System (NIPS)
NIPS detects and eliminates security violations by evaluating all the network traffic.

### 2. Host-Based Intrusion Prevention System (HIPS)
HIPS is software that monitors and analyzes suspicious activities for a host.

### 3. Network Behavior Analysis (NBA)
NBA is a type of IPS that detects and blocks unusual traffic flows and Denial of Service (DoS) attacks on the network.

### 4. Wireless Intrusion Prevention System (WIPS)
WIPS is a type of IPS that monitors and analyzes wireless network protocol traffic of wireless devices in a network.

## What Log Data Does IPS Have?

| Field | Description |
|-------|-------------|
| Date/Time | When the event occurred |
| Message | Details about the attack |
| Source IP Address | Where the traffic came from |
| Source Port | Source port number |
| Destination IP Address | Where the traffic was going |
| Destination Port | Destination port number |
| Action | What the IPS did (block/alert) |
| Device Name | Which device logged the event |
